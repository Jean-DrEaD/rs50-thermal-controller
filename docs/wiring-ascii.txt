[2] CASCATA DE BUCKS (24V -> 12V -> 5V)
================================================================================

   24V ===> +---------------+
            |  HW-411 #1    | === 12V ===> Fan, Bobina relé
            |  LM2596       |          ===> +---------------+
            |  IN+ ●  ● OUT+|              |   HW-411 #2   |
            |  IN- ●  ● OUT-|              |   LM2596      |
            +---------------+              |   IN+ ● ● OUT+|...> 5V (ESP32)
                                           |   IN- ● ● OUT-|
                                           +---------------+

   ⚠ AJUSTE OS POTENCIÔMETROS COM MULTÍMETRO ANTES DE CONECTAR CARGA:
     • HW-411 #1: gire trimpot até OUT+ medir 12.0V ±0.1V
     • HW-411 #2: gire trimpot até OUT+ medir  5.0V ±0.05V

[4] CIRCUITO DO NTC (sensor de temperatura do eixo)
================================================================================

       3V3 (ESP32) o
                   |
                   |
                  +-+
                  | |  R = 100K 1% (resistor superior do divisor)
                  | |
                  +-+
                   |
                   +------> GPIO1 (ADC1_CH0)
                   |
                   |---||---  C = 100nF (filtro pra GND, próximo do MCU)
                   |       |
                  +-+      |
                  | |      |
                  | |  NTC 100K B3950 (sonda no eixo do motor)
                  +-+      |
                   |       |
                  GND-----GND

   FÓRMULA: T = 1 / (1/T0 + (1/B) * ln(R/R0))   onde R0=100K@25°C, B=3950

[6] FAN 12V 4-PIN PWM
================================================================================

   FAN PINOUT (padrão Intel):
     Pino 1 (preto)    -> GND
     Pino 2 (amarelo)  -> +12V
     Pino 3 (verde)    -> TACH        ⚠ NÃO CONECTAR — cortar e isolar!
     Pino 4 (azul)     -> PWM (GPIO5, 25 kHz, duty 40-100%)


[7] SAÍDA AC PARA O MOTOR (chaveada pelo relé)
================================================================================

   ODESC FFBeast            Relé Songle SLA-24VDC          Motor 15Nm
   +-----------+            +-------------------+           +--------+
   |  AC OUT   |----------->| COM (terminal C)  |           |        |
   |   FASE    |            |                   |           | FASE   |
   +-----------+            |  NC ●------------>|---------->|        |
                            |                   |           |        |
                            |  NO ●  (livre)    |           +--------+
                            +-------------------+
                                  ^
                                  | bobina
                                  |
                              MOSFET drain

   ⚡ FAIL-SAFE: bobina desligada -> COM-NC fechado -> motor LIGADO
              ESP energiza bobina -> COM-NC aberto  -> motor DESLIGADO
              ESP travado/queimado -> bobina sem energia -> motor continua OK


[8] WS2812 (LED RGB indicador)
================================================================================

   GPIO21 ----> DIN (1º LED da strip)
   5V     ----> VCC
   GND    ----> GND

   LED_COUNT define o modo:
     1     -> SINGLE       (cor = estado)
     3-4   -> BAR          (estado + temp + pwm)
     5+    -> THERMOMETER  (preenchimento vertical)
